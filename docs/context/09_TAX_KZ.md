## УСН — главный режим для нашей ЦА

### Когда применима упрощёнка

ИП на УСН платит налог **3% или 2% от выручки** (ставка зависит от города, см. ниже).

**Условия применимости УСН:**
- Выручка ≤ 600 000 МРП в год = 2.595 млрд ₸
- Ограничение по виду деятельности: разрешённые виды (большинство торговли, услуг, общепита, ремесла)
- ИП **на УСН не платит НДС**, если работает с физлицами (B2C) и не пересекает порог НДС

### Ключевое уточнение: B2C на УСН не переходит на ОУР

Если ИП:
- На упрощённом режиме (УСН)
- Деятельность из разрешённого списка
- Клиенты — физлица (B2C)

Тогда **автоматически он не переходит на ОУР, не становится плательщиком НДС**, продолжает на УСН независимо от выручки (в пределах порога УСН 2.595 млрд ₸).

**Порог НДС — отдельная история.** Касается тех, кто работает с ЮЛ на ОУР. К нашей основной ЦА (B2C-микробизнес) — не применяется.

## Самозанятый — заменяет отменённый патент

**Патент отменён 01.01.2026.** В Wiki, движке, расчётах, материалах — **везде заменять «патент» на «Самозанятый»**.

### Кто может быть самозанятым

- Физлицо, не оформившее ИП
- Доход не превышает определённый порог (уточнять текущий лимит — он меняется по годам)
- Только B2C-деятельность определённых видов

### Налоги самозанятого

- ЕСП (единый совокупный платёж) — фиксированный взнос, заменяет ИПН + ОПВ + соц. отчисления + ООСМС
- Размер ЕСП зависит от региона: для городов Астана/Алматы/Шымкент/областные центры — выше; для районных центров и сёл — ниже
- Точные суммы — уточнять по году

### Когда ИП лучше самозанятого

- Если планируется выход за порог самозанятого
- Если нужно работать с ЮЛ
- Если нужны наёмные сотрудники (самозанятый не может нанимать)
- Если важна репутация ИП (договоры, тендеры)

## Соц. отчисления и ОПВ — основные правила для расчёта ФОТ

**За каждого работника на полной ставке:**

База: оклад работника (брутто, до удержания налогов).

**С работника удерживается из оклада (ИПН + ОПВ):**
- ИПН — 10% от (оклад − налоговый вычет)
- Налоговый вычет = 1 МЗП = 85 000 ₸/мес
- ОПВ — 10% от оклада (но не больше 50 МРП в месяц как лимит)

**За работника платит работодатель (поверх оклада, 17.5%):**
- ОПВР 3.5%
- СО 5%
- ООСМС 3%
- СН 6%

**Пример расчёта ФОТ:**

Оклад 200 000 ₸:
- Удерживается из оклада: ИПН (200 000 − 85 000) × 10% = 11 500 + ОПВ 200 000 × 10% = 20 000 → работник на руки **168 500 ₸**
- Работодатель платит сверху: 200 000 × 17.5% = **35 000 ₸**
- Полная стоимость работника для бизнеса: **235 000 ₸/мес**

## Налоги по типам бизнеса

### Розничная торговля, услуги (наша основная ЦА)
- УСН 2%–3% (в зависимости от города)
- Без НДС, если B2C

### Общепит (рестораны, кафе, доннерные)
- УСН 2%–3%
- Без НДС, если B2C
- Если работа с алкоголем — отдельная лицензия + акцизы

### Услуги (барбер, маникюр, массаж, спа)
- УСН 2%–3%
- Без НДС
- Если мастер на проценте без оклада — оформление через ИП-договор подряда (форма самозанятости)

### Образование (Академия, языковые школы, репетиторы)
- УСН 2%–3%
- Если частная школа с лицензией — отдельные правила
- Возможны льготы по социальным программам (для подачи на гранты)

## Что меняется при достижении порогов

- **Превысил 2.595 млрд ₸ выручки** → переход на ОУР, обязательная регистрация по НДС
- **Нанял первого работника** → возникают обязательства по ОПВР, СО, ООСМС, СН (17.5% сверх оклада)
- **Начал работать с ЮЛ на ОУР с НДС** → может потребоваться добровольная регистрация по НДС, если контрагент требует НДС-счёт-фактуру

## Правила использования налогов в продуктах ZEREK

- В Wiki-обзорах — общая информация, без сложных расчётов: «УСН 3% для Актобе», «При найме сотрудника — нагрузка 17.5%»
- В Quick Check — точный расчёт для конкретного формата + города + типа клиентов (B2C/B2B)
- В FinModel — полный налоговый блок с разбивкой по месяцам
- В BizPlan — налоговый раздел в банковском формате с обоснованием режима

## Что НЕ забывать

- Ставки **меняются ежегодно**. МРП — каждый год новый. Не использовать «вечные значения».
- УСН в городах **бывает разная** — 2% и 3%. Не путать.
- НДС с физлицами **не платится** — не путать массового клиента с большим B2B.
- При найме первого работника — нагрузка **резко возрастает**. Закладывать в стартовый бюджет.
- Самозанятый ≠ ИП. Это разные правовые статусы с разными ограничениями.
